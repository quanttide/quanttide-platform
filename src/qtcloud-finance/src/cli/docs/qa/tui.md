### BUG-002 确认按钮功能失效

| 字段 | 内容 |
|------|------|
| 严重程度 | 中 |
| 状态 | 待确认 |
| 模块 | TUI 交互流程 |

**描述**：输入账单后内容已自动保存，右侧确认按钮形同虚设。

**复现步骤**：
1. 启动 TUI 应用
2. 输入消费描述（如"早餐 10 元"）
3. 观察是否自动写入账本

**预期结果**：应等待用户确认后才写入账本。

**实际结果**：账单已自动写入，确认按钮无实际作用。

### BUG-003 Y/N 快捷键提示多余

| 字段 | 内容 |
|------|------|
| 严重程度 | 低 |
| 状态 | 待确认 |
| 模块 | TUI 交互提示 |

**描述**：对话面板中提示"按 Y 确认写入，或按 N 重新生成"，但实际已自动保存。

**复现步骤**：
1. 启动 TUI 应用
2. 输入消费描述
3. 查看对话面板提示信息

**预期结果**：如已自动保存，不应显示确认提示。

**实际结果**：仍显示 Y/N 操作提示。
