# 产品边界说明

量潮商务云与量潮销售云的功能边界定义。

## 边界定义

| 维度 | 销售云 | 商务云 |
|------|--------|--------|
| **阶段** | 前端获客 | 后端成交 |
| **核心动作** | 把客户找到 | 把钱收进来 |
| **对应场景** | BRD 场景 1/2/3 | BRD 场景 4 |
| **关注对象** | 人（潜在客户） | 单（报价/合同） |

## 职责划分

### 销售云

- 获客渠道管理（多平台案例发布）
- 信任背书（案例库、素材附件）
- 潜客归集与管理
- 跟进记录

### 商务云

- 报价制定（方案模板、报价单）
- 合同管理（模板、签署流程）
- 合同履约跟踪

## 协作关系

```
潜在客户 → 销售云（归集） → 商务云（报价） → 合同签署 → 成交
```

销售云负责把客户"找到"，商务云负责把钱"收进来"。
