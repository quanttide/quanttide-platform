# 产品边界说明

qtcloud 产品组合的边界定义。

## 产品组合

| 产品 | 核心能力 | 子仓库 |
|------|----------|--------|
| **销售云** | 获客与客户管理 | qtcloud-sales |
| **商务云** | 报价与合同管理 | qtcloud-business |
| **思考云** | 思维收集与澄清 | qtcloud-think |
| **写作云** | 写作辅助 | qtcloud-write |
| **健康云** | CBT 工具 | qtcloud-health |
| **数字资产云** | 资产管理 | qtcloud-asset |
| **连接云** | 智能体协作 | qtcloud-connect |

## 销售云与商务云边界

| 维度 | 销售云 | 商务云 |
|------|--------|--------|
| **阶段** | 前端获客 | 后端成交 |
| **核心动作** | 把客户找到 | 把钱收进来 |
| **对应场景** | BRD 场景 1/2/3 | BRD 场景 4 |
| **关注对象** | 人（潜在客户） | 单（报价/合同） |

### 销售云职责

- 获客渠道管理（多平台案例发布）
- 信任背书（案例库、素材附件）
- 潜客归集与管理
- 跟进记录

### 商务云职责

- 报价制定（方案模板、报价单）
- 合同管理（模板、签署流程）
- 合同履约跟踪

### 协作关系

```
潜在客户 → 销售云（归集） → 商务云（报价） → 合同签署 → 成交
```

销售云负责把客户"找到"，商务云负责把钱"收进来"。
