# 量潮写作云

这是系统视角对产品视角的 PRD 的修改思路。

## 核心定位

量潮写作云是一个**透明化 AI 写作过程**的工具，帮助作者**理解和掌握构思方法**，实现**能力内化**与**创作主权**回归。

## 做什么（能力范围）

- 透明化展示 AI 如何构建文章结构（核心观点提取、论点展开逻辑、段落组织依据）—— **构思审计**
- 透明化展示 AI 如何优化内容（表达改进理由、风格调整说明、读者视角推演）—— **段落评审**
- 透明化展示 AI 如何迭代作品（版本对比、改动权衡、效果验证）—— **渐进完善**
- 引导用户按“看懂 → 会用 → 掌控”路径进阶 —— **能力进阶**
- 让用户**主动掌控创作**，每次协作都是学习，写作能力同步增长

## 不做什么（能力边界）

- 不做**黑箱生成**（传统工具：用户被动接受成品）
- 不做单纯的**内容生成**（不只是生成内容，而是展示创作思考过程）
- 不剥夺**创作主权**（用户拥有完整理解，表达个人独特视角）
- 不强制**统一风格**（解决“AI输出风格统一，无法表达个人视角”的痛点）

## 方法论基础

- **正向应用**：一句话 → 一段概述 → 一页大纲 → 完整作品
- **反向应用**：混乱草稿 → 提炼核心 → 重组结构 → 拆分细节 → 清晰有力

## 核心价值

> 白盒展示构思过程，用户主动掌控创作。每一句话的组织逻辑清晰可见，告别内容空洞。能力内化，重新掌握写作主动权。
