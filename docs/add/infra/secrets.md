# 密钥管理

使用 HashiCorp Vault 提供统一密钥服务，本地和云端采用适配各自环境的存储方案，对研发保持一致的 API 接口。

## 架构

```
业务层 (平台A/B、CI/CD)
    ↓
Vault 层 (加密微服务)
    ↓
存储层：本地 Raft / 云端 S3 兼容存储
```

## 设计要点

**Vault 层**
- 标准信封加密（AES-256-GCM + RSA-4096/OAEP），密文格式公开
- 无状态密码学引擎，存储后端可切换

**存储层**
- 本地环境：Raft 内置存储，数据写入本地磁盘
- 云端环境：S3 后端直写 OSS/COS，利用跨云同步容灾
- 主密钥：对接云 KMS 实现 Auto-unseal

## 优势

- 研发侧接口统一，无需关心底层存储差异
- 无数据库依赖，运维成本低
- 标准加密协议，数据可迁移
