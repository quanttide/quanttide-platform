# quanttide-platform

量潮应用系统主仓库。

## 仓库结构

```
quanttide-platform/
├── apps/qtcloud/       → qtcloud 产品线主仓库
├── apps/qtadmin/       → qtadmin 管理后台
├── apps/qtdata/        → qtdata 数据服务
├── apps/qtclass/      → qtclass 课堂服务
├── apps/qtcloud-*     → 独立产品模块
└── apps/qtcloud-knowl → 知识云
```

→ [产品边界说明](docs/index.md)

版本策略、设计原则等项目约定见 [CONTRIBUTING.md](CONTRIBUTING.md#项目约定)。
