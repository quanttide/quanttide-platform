# 量潮应用系统架构文档

量潮应用系统（`qtapps`），是量潮科技旗下软件应用体系的总称。

## 仓库层级

```
quanttide-platform/
├── apps/qtcloud/     → qtcloud (组合所有 qtcloud-*)
├── apps/qtadmin/     → qtadmin
├── apps/qtdata/      → qtdata
├── apps/qtclass/     → qtclass
└── apps/qtcloud-*   → 独立产品
```

→ [仓库层级说明](docs/index.md)
